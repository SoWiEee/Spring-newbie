API Reference

/v1/add_user

HTTP Method: POST

Description: Add a new user. Requires name, email, and password in the request body.

/v1/get_user

HTTP Method: GET

Description: Retrieve user information using the email provided in the request header.

/v1/fix_user

HTTP Method: PUT

Description: Update user information. Requires email in the request header and new data in the request body.

/v1/delete_user

HTTP Method: DELETE

Description: Delete a user. Requires email and password in the request header.
