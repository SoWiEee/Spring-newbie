Add User

Add a new user. Requires name, email, and password in the request body.

Request

POST /v1/add_user

Body - { "name": "String", "e-mail": "String", "password": "Hashed String, MD5" }

Response

Body - { "rsp_code": Integer response code, "rsp_msg": "String description of rsp" }

rsp_code - 20(User added)；21(Email already exists)；30(Incorrect params)；31(Missing params)

Get User

Retrieve user information using the email provided in the request header.

Request

GET /v1/get_user

Header - Email of user

Response

Body - { “rsp_code”: Integer response code, “rsp_msg”: “String description of rsp”, “data”: { “name”: “String”, “email”: “String” } }

rsp_code - 20(User fetched)；30(Incorrect params)；31(Missing params)；40(Email not found)

Fix User

Update user information. Requires email in the request header and new data in the request body.

Request

PUT /v1/fix_user

Header - Email of updated user

Body - { “new_name”: “String, null if no update”, “new_e-mail”: ”String, null if no update”, “password”: “Hashed String, MD5” }

Response

Body - { "rsp_code": Integer response code, "rsp_msg": "String description of rsp" }

rsp_code - 20(User updated)；30(Incorrect params)；31(Missing params)；40(Email not found)；50(Incorrect password)

Delete User

Delete a user. Requires email and password in the request header.

Request

DELETE /v1/delete_user

Header - Email of deleted user, Password of deleted user

Response

Body - { "rsp_code": Integer response code, "rsp_msg": "String description of rsp" }

rsp_code - 20(User deleted)；30(Incorrect params)；31(Missing params)；40(Email not found)
